Eigen Website Coderen

Voor het vak Internetstandaarden heb ik mijn eigen website gecodeerd. Deze website was gebaseerd op het ontwerp dat ik eerder had gemaakt tijdens het vak Content.

Ik moest het ontwerp vertalen naar echte HTML- en CSS-code, zodat het als een werkende website op internet zou functioneren. Hierbij moest ik goed letten op de regels van HTML en CSS, en zorgen dat er geen fouten of bugs in de code zaten.

Tijdens het coderen kwam ik verschillende uitdagingen tegen, zoals het juist plaatsen van elementen, het gebruik van responsive design en het oplossen van fouten in de console.

Uiteindelijk is het gelukt om mijn eerste eigen website helemaal zelf te bouwen. Dit project heeft mij veel geleerd over structuur, techniek en het toepassen van een ontwerp in echte code.
