Tamagotchi Game

Voor het vak Inleiding Programmeren heb ik een eigen game gemaakt met JavaScript. In deze opdracht leerde ik hoe je interactieve elementen toevoegt aan een website, zoals animaties en knoppen waarmee je iets kunt besturen.

Ik heb een Tamagotchi-achtige game ontworpen waarin je voor een virtuele hamster moet zorgen. Deze hamster heb ik zelf ontworpen in Illustrator. Je moet hem op tijd eten geven, wassen, laten slapen en blij houden.Als je dat niet doet, dalen de statistieken van de hamster. Wanneer een van de stats op 0 komt, gaat het slecht met je Tamagotchi.

Door dit project heb ik niet alleen beter leren programmeren met JavaScript, maar ook geleerd hoe je visueel ontwerp en code met elkaar combineert om iets speels en interactiefs te maken.
